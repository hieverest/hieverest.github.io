    Defining and using Arrow Functions

    In JavaScript, functions may be anonymously defined using the “arrow” (=>) syntax, which is sometimes referred to as a lambda expression due to Common Lisp similarities.

    This syntax is primarily intended for lightweight callbacks where inline definitions are more convenient, as opposed to the heavyweight syntax, characteristic of regular anonymous functions.
